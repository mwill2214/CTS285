# Flask Howto
https://flask.palletsprojects.com/en/stable/

Using Anaconda:
- launch Anaconda Prompt
- launch Spyder

In Anaconda prompt:
pip install flask

if this completes with no error, we're good, continue.

In Spyder
put in the test python code
try to run 
if it doesn't run, go back to prompt, go to the same folder, and type:
flask run app.py
